# Surface OPENGL
Realisation de surface de Bezier, Catmull-Rom & B-Spline uniforme bi-cubique et BSpine non uniforme de degrès quelconque. Nécéssite Armadillo.

# Interraction clavier + menu souris clique centrale.
-Affichage des courbes
- Gestion des points de controles : Mode selection point (Menu) + déplacement fleche directionnelle
- Gestion du vecteur nodal : Mode selection vecteur Nodal (Inverse de selection point) + deplacement fleche directionnelle
- Mouvement des points de controles
- Changement d'ordre
- Changement d'outil de selection des points de controles : Individuel, ligne vecticale ou horizontale
