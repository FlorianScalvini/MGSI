# Surface Implicite 

Code en lien avec les surfaces implicite réalisé en C++ - OpenGL

## Différentes possibilités implémenté

- Choix fonction de potentiel : Wyvill - Nishimura - Murakani 
- Modification des parametres : Seuil & Rayon d'influence 
- Composition - Union - Intersection - Différence (Par rapport au premier point)
